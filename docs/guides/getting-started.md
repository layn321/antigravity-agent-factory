# Getting Started with Antigravity Agent Factory

Welcome to the **Antigravity Agent Factory**, a high-fidelity system for building verifiable, purposeful, and safe AI agentic systems using the Axiomatic Agentic Framework.

## 1. Explore Core Concepts
- **Agents**: Specialized AI entities with defined roles.
- **Skills**: Deterministic "recipes" for specific tasks.
- **Workflows**: Multi-step processes (e.g., Feature Development, Bug Fix).
- **Axioms**: The foundational "laws" (Love, Truth, Beauty) enforced by the Integrity Guardian (L0).

## 2. Your First Steps
To begin using the Factory, we recommend the following flow:
1.  **Read the README**: [README.md](../../README.md) for a high-level overview.
2.  **Review the SDLC Guide**: [docs/guides/sdlc-usage-guide.md](sdlc-usage-guide.md) to understand how to move through development phases.
3.  **Run a Quickstart**: Use the CLI to generate a demo project:
    ```powershell
    python cli/factory_cli.py --quickstart
    ```
