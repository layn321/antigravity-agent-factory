# Factory Architecture Hierarchy

This document maps the relationships between workflows, agents, skills, and knowledge across the 7 SDLC Phases.

## 01 - Ideation
- **Workflows**: `research`, `alpha-factor-mining`
- **Agents**: System Architect Steward, Knowledge Operations
- **Skills**: `framing-problems`, `brainstorming-ideas`

## 02 - Requirements
- **Workflows**: `feature-development`, `plane-task-management`
- **Agents**: System Architect Steward
- **Skills**: `managing-plane-tasks`, `writing-prd`

## 03 - Architecture
- **Workflows**: `ai-system-design`
- **Agents**: System Architect Steward
- **Skills**: `designing-ai-systems`

## 04 - Build
- **Workflows**: `antigravity-factory-building`, `agent-development`, `fastapi-api-development`
- **Agents**: Python AI Specialist, Code Integrity Guardian
- **Skills**: `developing-fastapi`, `developing-nextjs`

## 05 - Test & Eval
- **Workflows**: `agent-testing`, `coordination-testing`, `quality-gate`
- **Agents**: Test Conductor
- **Skills**: `testing-frontend`, `clean-code-review`

## 06 - Deploy
- **Workflows**: `azure-deployment`, `release-management`
- **Agents**: Release Engineer
- **Skills**: `committing-releases`

## 07 - Monitor
- **Workflows**: `incident-response`, `prompt-iteration`
- **Agents**: Cognitive Cycle Engineer
- **Skills**: `prompt-engineering`
