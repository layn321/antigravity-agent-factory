# Planning Routine

This workflow allows the agent to decompose high-level dashboard requirements into actionable technical tasks.

## Steps

1. **Decomposition**
   - Break down FEATURE requests into:
     - Data sourcing tasks.
     - Analysis tasks.
     - UI/Visualization tasks.

2. **Estimation**
   - Apply Story Points (1, 2, 3, 5, 8).
   - Use the `managing-plane-tasks` skill to estimate based on complexity.

3. **Scheduling**
   - Assign tasks to cycles/sprints.
   - Define start and target dates.

4. **Registry**
   - Create the issues in Plane using the MCP tool.
   - Link tasks to the 'statistical dashboard' module.
