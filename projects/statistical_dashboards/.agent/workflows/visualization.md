# Visualization Blueprint

This workflow ensures all visualizations generated by agents meet professional, presentation-ready standards.

## Principles

1. **Clarity**: Minimize non-data ink. Use clean labels and legends.
2. **Context**: Every chart must have a title, axis labels, and a brief interpretation.
3. **Consistency**: Use the project's brand colors (defined in `theme.json`).

## Steps

1. **Selection**
   - Choose chart type based on data:
     - Trends -> Line Chart.
     - Distributions -> Histogram/Density Plot.
     - Relationships -> Scatter Plot.
     - Categories -> Bar Chart.

2. **Generation**
   - Use `plotly.express` for interactive HTML exports.
   - Use `matplotlib` for static PDF/Markdown embeds.
   - Ensure `tight_layout()` is called.

3. **Refinement**
   - Add annotations for key data points (max, min, outliers).
   - Export to `plots/visuals/`.

4. **Integration**
   - Embed visualization in the relevant report or dashboard view.
